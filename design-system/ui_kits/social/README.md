# RISE — Social & Flyer Kit

Ready-to-adapt templates for social graphics and print. Open `index.html` for the gallery.

## Templates
- **Instagram post (1:1)** — announcement with gold-glow accent. Export 1080×1080.
- **Story (9:16)** — full ascent gradient, logo + CTA. Export 1080×1920.
- **Quote post (1:1)** — testimonial on dark charcoal. Export 1080×1080.
- **Event flyer / poster (4:5)** — photo + details + register CTA. Export 4:5 or scale to A5/A4.
